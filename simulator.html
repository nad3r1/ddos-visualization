Important Context

Simulator Limitations

Educational Model

The simulator does not use live packet captures, SIEM alerts, flow telemetry, or real monitoring data. All values are simplified educational estimates designed to show relative differences between attack and mitigation pairs.

One Primary Control At A Time

Each run models one primary mitigation so the relationship between attack mechanism and defense fit stays easy to explain. Real DDoS defense is usually layered across host, application, edge, and upstream infrastructure.

Real Incidents Are Broader

In real incidents, upstream provider support, network topology, available bandwidth, service capacity, and application design all influence the outcome. This simulator isolates the core teaching concepts rather than reproducing full production complexity.
